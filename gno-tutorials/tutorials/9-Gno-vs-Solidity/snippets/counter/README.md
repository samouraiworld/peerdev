# Counter

Basic counter implementation showing state management and incrementing values.

**Key Difference:** Solidity uses `uint256` with public state variables, Gno uses `int` with getter functions.

## Testing

Run from the tutorial root directory:
```bash
# Test Gno version
./test-snippets.sh 1 gno

# Test Solidity version
./test-snippets.sh 1 sol
```
