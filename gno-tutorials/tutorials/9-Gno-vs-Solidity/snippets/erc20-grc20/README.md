# ERC20-GRC20

Token standard implementation for fungible tokens.

**Key Difference:** Solidity uses ERC20 standard, Gno uses GRC20 with `grc20` package and `avl.Tree` for balances.

## Testing

Run from the tutorial root directory:
```bash
# Test Gno version
./test-snippets.sh 2 gno

# Test Solidity version
./test-snippets.sh 2 sol
```
